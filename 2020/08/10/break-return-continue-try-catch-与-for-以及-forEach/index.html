for 循环中的 continue

for 循环中的 break

for 循环中的 return

小结

  • break: 终止循环（跳出循环），不再执行循环体下面的代码；
  • return: 终止函数（&& 终止循环），不再执行循环体下面的代码 && 不再执行循环体外下方代码；
  • continue：终止当前这一次的循环，进行下一次循环；

forEach 循环中的 return, try…catch…

forEach 循环中的 return

forEach 方法说明

  • return 在里面的作用是终止本次循环的函数执行（功能上类似于， for 循环中 continue 的使用）

forEach 循环中的 try…catch…

小结

  • return: 中断本次循环直接进入下次循环；
  • try…catch…: 跳出整个循环；
