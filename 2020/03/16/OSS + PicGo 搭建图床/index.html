OSS + PicGo 搭建图床
2020-03-16
字数总计: 472|阅读时长: 1 分钟

前言

作者在搭建个人博客时，采用 Gulp 压缩图片来提高网站的加载速度，但是发现加载图片还是很慢。
后来经大佬的指导，作者将图片放到图床明显的提高了网站的加载速度。
所以作者记录下来 OSS + PicGo 搭建图床的方法，希望可以给需要的小伙伴一个帮助。
（Ps: 当前博客的背景图7M多）

阿里云OSS过程

注册阿里云账号

从控制台搜索找到OSS并开启服务(作者在这个地方购买了一年40G的存储空间)

创建存储空间

链接： https://help.aliyun.com/document_detail/31885.html

  • 进入对象存储 OSS，点击 Bucket 列表中自己新建的 bucket

    PS：读写权限选择公共读，其他可以看需求

新建授权策略

返回控制台，搜索访问控制点击进入(类似第一步操作)

  • yourBucketName 换成你的 bucket 名字

新建用户

  • 点击用户，新建用户

  • 成功后，记录下自己的 ID 跟密码

解释

  • KeyId: 前面步骤记录的 id
  • KeySecret: 前面步骤记录的 secret
  • 存储空间名：前面步骤创建的 bucket 名字
