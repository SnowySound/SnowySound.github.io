gittalk 评论添加 err notfound 的解决办法
2020-01-16
字数总计: 182|阅读时长: 1 分钟

前言

在使用 Hexo + GitHub 搭建这个博客的时候，在添加 gittak 评论系统时， 出现了 err notfound 的错误。记录下来。

为评论新建一个仓库 && 初始化 issue

在 GitHub Apps 的 OAuth Apps 新建 OAuth App

1）Application name 可以随便填一个。
2）Homepage URL 必须是博客仓库的域名（https://github.com/SnowySound/SnowySound.github.io）。
3）Authorization callback URL 必须是博客的域名（https://www.snowysound.github.io/）。

主题配置文件中配置

1）owner 为 GitHub 的登录名
2）repo 为 第一步中新建的评论仓库
3）clientID 和 clientSecret 则为 OAuth Apps 生成的

最后

如有问题，添加评论，我看到后会给答复（大家也可以在评论区中互相讨论）。
