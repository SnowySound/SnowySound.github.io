前言

在开发一些商品售卖的网站时，我们经常会遇到这样的需求： 放大商品的局部从而使消费者可以很轻松的观察商品的局部特征。

实现代码

代码解释

放大镜的实现原理：通过鼠标移动来实现，选中的小部分可以放大展示在屏幕上

主要的方法：

  • onmouseout
  • onmouseover
  • onmousemove

其他

放大镜实现的核心代码就是计算当前所处的位置是否是需要放大的地方。

这就要求需要计算(以横轴为例)：

  • offsetLeft:子元素相对父元素的左边距离
  • offsetWidth:盒子的宽度，不报括滚动条
  • event.clientX:鼠标的X轴，相对于整个页面

当当前页面的需放大部分据页面左边还存在其他标签的影响时，需要减去这些标签的宽度 或者是 offsetLeft
