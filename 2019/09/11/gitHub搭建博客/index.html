Hexo 和 Github 搭建博客
2019-09-11
字数总计: 496|阅读时长: 1 分钟

前言：

使用 GitHub 搭建博客有很多优点：

  1. 静态页面渲染，访问速度快
  2. Git 版本库存储，可以恢复之前文件，数据安全
  3. 可以采用自己的域名搭建博客，充分实现个性化

前期配置：

1、官网安装 node.js。
2、安装Git和配置好Git环境

安装成功的象征就是

  • 在电脑上任何位置鼠标右键能够出现两个选择(Git GUI here && Git Bash here)
  • git --version
// 显示安装的 git 版本
3、注册Github账户，Github地址
4、安装Hexo
5、安装依赖

7、下载主题，在_config.yml文件中修改主题路径，实现个性化配置
