Window.location 获取 url 各部分的值

URL(Uniform Resource Locators) 一个普通的 url 的组成部分

  • 以 http://www.baidu.com:8080/item/%E6%96%87%E4%BB%B6/6270998?fr=aladdin#name 为例
  • 如上，可以清楚看到一个 url 的组成：
  • 协议： http or https
  • 域名： www.baidu.com
  • 端口号： 8080
  • 路径(文件地址)： /item/%E6%96%87%E4%BB%B6/6270998 第一个 / 跟第一个 ？ 或 # 之间
  • 查询(参数)：?fr=aladdin 第一个 ？ 与 #之间
  • 锚点： #name # 后面的值
