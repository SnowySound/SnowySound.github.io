Websocket 的 ws，wss 与 http,https

  • ws 对应 http
  • wss 对应 https

Websocket 的用法

new Websocket(url, [,protocols])

  • protocols (可选):
    一个协议字符串或者一个包含协议字符串的数组。这些字符串用于指定子协议，这样单个服务器可以实现多个WebSocket子协议（例如，您可能希望一台服务器能够根据指定的协议（protocol）处理不同类型的交互）。如果不指定协议字符串，则假定为空字符串。
  • SECURITY_ERR (抛出异常)：
    尝试连接的端口被阻止。
